# Sidebar Menu UI

This project is a responsive sidebar navigation menu created using **HTML** and **CSS**.  
It features a smooth sliding animation, social media icons, and a clean UI layout similar to modern web applications.

# Features
- Hamburger icon to toggle the sidebar
- Smooth slide-in and slide-out animation
- Navigation links with icons
- Social media section at the bottom
- Transparent glass-effect background
- Fully responsive design

# Technologies Used
- HTML5
- CSS3
- Font Awesome Icons

# How It Works
- Clicking the menu icon (`fa-bars`) opens the sidebar
- The close icon (`fa-close`) hides the sidebar
- Uses CSS transitions and a checkbox trick for toggle
